O que são conjuntos disjuntos?

- São conjuntos que não têm nenhum elemento em comum. Logo, a operação de interseção entre eles, resultaria em um conjunto vazio, {}.

Defina as seguintes operações “clássicas” entre conjuntos, usando apenas notação matemática:

- União:  {x | x ∈ A ou x ∈ B}
- Interseção:  {x | x ∈ A e x ∈ B}.
- Diferença: {x | x ∈ A e X ∉ B}
- Produto Cartesiano: {(x, y) | x ∈ A e y ∈ B }

Explique o que é Conjunto das Partes de um conjunto A, denotado por ℘(A), depois escreva ℘(A) quando:

Conjunto das Partes são todos os subconjuntos formados a partir dos elementos de determinado conjunto. Não esquecendo do onipresente conjunto vazio.

A = { a , b } =>  ℘(A)= {{ }, {a}, {b}, {a, b}}
A = { 1, 2, 3 } => ℘(A)= {{ }, {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3}}
